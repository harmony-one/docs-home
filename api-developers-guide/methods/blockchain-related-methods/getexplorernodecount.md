# node-count

Returns number of explorer nodes in the network.

#### Returns

* `Integer` - Number of explorer nodes in the network.

#### Sample Curl Request

```bash
curl -H "Content-Type:application/json" -X GET "e1.b.hmny.io:5000/node-count"
```

#### Sample Curl Response

```javascript
2
```
