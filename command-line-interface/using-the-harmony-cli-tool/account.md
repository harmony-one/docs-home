# Account

## Available actions

### Get balance on account

Pass the address of an account to `account` to get the account's balance.

**Example:**

```bash
$ ./hmy_cli account one103q7qe5t2505lypvltkqtddaef5tzfxwsse4z7
41.714285714
```
